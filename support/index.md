# Stringly Support

Need help with Stringly? Email **[kevinliu217@gmail.com](mailto:kevinliu217@gmail.com)** — replies within 48 hours.

## Common questions

**My TestFlight install crashed**
Delete the app, reinstall, try again. If it still crashes: email me with your device model + iOS version + the last screen you saw.

**My logged data is gone**
Stringly stores everything on-device. If you reinstalled the app, the data is gone (cloud sync is on the M2 roadmap — coming soon).

**I want a feature**
Send it via the in-app **Send feedback** button (Profile → Settings → Send feedback) or email me directly.

**I found a bug**
In the app: **Profile → Settings → Report a bug**. Or email kevinliu217@gmail.com.
**Valid bug reports earn one free month of Pro** once the Pro tier ships in M2.

**Why does the app ask for camera / photo permissions?**
Camera is used to scan racket serial numbers (T-joint OCR). Photos are used so you can attach a racket portrait. Both are optional — the app works without them.

**How do I delete all my data?**
Settings → Privacy → Delete all data. This wipes the local database and resets the app to first-launch state.
